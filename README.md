# Резюме

Адаптивное веб-резюме с тёмной и светлой темами, разработанное на чистом HTML, CSS и JavaScript.

## Функциональность

- **Адаптивный дизайн** - корректное отображение на мобильных устройствах и десктопах
- **Переключение темы** - тёмная и светлая темы с сохранением выбора пользователя
- **Интерактивное модальное окно** для быстрой связи

## Стек

- **HTML5** - семантическая разметка
- **CSS3** - кастомные свойства (CSS Variables), Flexbox, Grid, медиа-запросы
- **JavaScript**

## Ключевые разделы

- **Профиль** - фото, контакты, основная информация
- **Обо мне** - краткое описание и цели
- **Навыки** - список технических компетенций
- **Опыт работы** - история трудоустройства
- **Образование** - высшее и дополнительное образование

## Адаптивность

- **Мобильные устройства** (< 1200px) - вертикальная компоновка
- **Десктоп** (≥ 1200px) - горизонтальная двухколоночная layout
